## NARR-01: Concurrent OCC Collision

### Scenario Description
Two [`CreditAnalysisAgent`](ledger/agents/credit_analysis_agent.py) instances start simultaneously on the same application after documents are processed. Both read the credit stream at version 0 (only [`CreditRecordOpened`](ledger/schema/events.py) exists).

### Expected Behavior
1. Agent A appends [`CreditAnalysisCompleted`](ledger/schema/events.py) at `expected_version=0` → **succeeds** → stream version becomes 1
2. Agent B hits [`OptimisticConcurrencyError`](src/event_store.py), reloads stream, sees Agent A's result
3. Agent B confirms analysis is still needed (it is — same application)
4. Agent B appends its own [`CreditAnalysisCompleted`](ledger/schema/events.py) at `expected_version=1` → **succeeds** → version becomes 2
5. Both agents complete without raising to caller

### Test Assertions
```python
# tests/test_narratives.py::test_narr01
async def test_narr01_concurrent_occ_collision(event_store, registry_client):
    # Setup: Create application NARR-01 with documents processed
    app_id = "APEX-NARR01"
    company_id = "COMP-031"  # Manufacturing, MEDIUM risk, ~$3.8M revenue
    
    # Create credit stream with CreditRecordOpened only
    await event_store.append(f"credit-{app_id}", [
        CreditRecordOpened(application_id=app_id, company_id=company_id)
    ], expected_version=0)
    
    # Start two agents concurrently
    agent_a = CreditAnalysisAgent(event_store, registry_client)
    agent_b = CreditAnalysisAgent(event_store, registry_client)
    
    state_a = {"application_id": app_id, "company_id": company_id, "expected_version": 0, ...}
    state_b = {"application_id": app_id, "company_id": company_id, "expected_version": 0, ...}
    
    # Run concurrently
    results = await asyncio.gather(
        agent_a.process(state_a),
        agent_b.process(state_b),
        return_exceptions=False  # Both must complete without raising
    )
    
    # Assertions
    credit_stream = await event_store.load_stream(f"credit-{app_id}")
    credit_events = [e for e in credit_stream if e.event_type == "CreditAnalysisCompleted"]
    
    assert len(credit_events) == 2, "Exactly 2 CreditAnalysisCompleted events"
    assert credit_events[0].stream_position == 1
    assert credit_events[1].stream_position == 2
    assert credit_events[1].metadata.get("causation_id") is not None, "Second event has causation"
```

### Implementation Requirements
- [`CreditAnalysisAgent.write_output()`](ledger/agents/credit_analysis_agent.py) must use `append_with_retry()`
- Retry logic: catch [`OptimisticConcurrencyError`](src/event_store.py), reload stream, re-execute, retry append
- Max retries: 5 with exponential backoff (10ms, 20ms, 40ms, 80ms, 160ms)
- Both agents must complete successfully (no exceptions raised to caller)

## NARR-02: Document Extraction Failure (Missing EBITDA)

### Scenario Description
[`DocumentProcessingAgent`](ledger/agents/stub_agents.py) processes an application where the income statement PDF has no EBITDA line item (one of the 4 PDF variants generated by [`datagen/pdf_generator.py`](datagen/pdf_generator.py)).

### Expected Behavior
1. Week 3 extraction pipeline returns `facts.ebitda = None`
2. [`ExtractionCompleted`](ledger/schema/events.py) event has:
   - `facts.ebitda = None`
   - `field_confidence["ebitda"] = 0.0`
   - `"ebitda"` in `extraction_notes`
3. [`QualityAssessmentCompleted`](ledger/schema/events.py) event has:
   - `"ebitda"` in `critical_missing_fields`
4. [`CreditAnalysisAgent`](ledger/agents/credit_analysis_agent.py) receives quality flags
5. Credit agent notes `data_quality_caveats` in output
6. Credit agent caps `confidence` at 0.75
7. **Application continues** — not blocked

### Test Assertions
```python
# tests/test_narratives.py::test_narr02
async def test_narr02_missing_ebitda(event_store, registry_client):
    app_id = "APEX-NARR02"
    company_id = "COMP-044"  # Healthcare, STABLE, missing_ebitda variant
    
    # Run DocumentProcessingAgent
    doc_agent = DocumentProcessingAgent(event_store)
    state = {
        "application_id": app_id,
        "company_id": company_id,
        "document_paths": [
            f"documents/{company_id}/income_statement_2024_missing_ebitda.pdf",
            f"documents/{company_id}/balance_sheet_2024.pdf"
        ],
        ...
    }
    await doc_agent.process(state)
    
    # Load docpkg stream
    docpkg_stream = await event_store.load_stream(f"docpkg-{app_id}")
    extraction_event = next(e for e in docpkg_stream if e.event_type == "ExtractionCompleted")
    quality_event = next(e for e in docpkg_stream if e.event_type == "QualityAssessmentCompleted")
    
    # Assertions on extraction
    assert extraction_event.payload["facts"]["ebitda"] is None
    assert extraction_event.payload["field_confidence"]["ebitda"] == 0.0
    assert "ebitda" in extraction_event.payload["extraction_notes"]
    
    # Assertions on quality
    assert "ebitda" in quality_event.payload["critical_missing_fields"]
    
    # Run CreditAnalysisAgent
    credit_agent = CreditAnalysisAgent(event_store, registry_client)
    credit_state = {"application_id": app_id, "company_id": company_id, ...}
    await credit_agent.process(credit_state)
    
    # Load credit stream
    credit_stream = await event_store.load_stream(f"credit-{app_id}")
    credit_event = next(e for e in credit_stream if e.event_type == "CreditAnalysisCompleted")
    
    # Assertions on credit decision
    assert credit_event.payload["decision"]["confidence"] <= 0.75
    assert len(credit_event.payload["decision"]["data_quality_caveats"]) > 0
```

### Implementation Requirements
- Week 3 pipeline integration in [`DocumentProcessingAgent._node_extract_income_statement()`](ledger/agents/stub_agents.py)
- Handle `None` values from extraction — do NOT default to zero
- Quality assessment LLM call in [`_node_assess_quality()`](ledger/agents/stub_agents.py)
- [`CreditAnalysisAgent`](ledger/agents/credit_analysis_agent.py) must read quality flags and adjust confidence

## NARR-03: Agent Crash and Recovery

### Scenario Description
[`FraudDetectionAgent`](ledger/agents/stub_agents.py) starts processing and crashes after the `load_facts` node (simulated in test by calling `agent._simulate_crash_after_node("load_facts")`).

### Expected Behavior
1. First session: Agent executes `load_facts` node, records [`AgentNodeExecuted`](ledger/schema/events.py), then crashes
2. [`AgentSessionFailed`](ledger/schema/events.py) event appended with:
   - `recoverable = True`
   - `last_successful_node = "load_facts"`
3. New [`FraudDetectionAgent`](ledger/agents/stub_agents.py) instance starts
4. [`reconstruct_agent_context()`](ledger/integrity/gas_town.py) reads crashed session stream
5. Identifies `load_facts` completed
6. New session starts with `context_source = "prior_session_replay:{crashed_session_id}"`
7. Recovery resumes from `cross_reference_registry` node (skipping `load_facts`)
8. [`AgentSessionRecovered`](ledger/schema/events.py) event in new session stream
9. **No duplicate `load_facts` work**

### Test Assertions
```python
# tests/test_narratives.py::test_narr03
async def test_narr03_crash_recovery(event_store, registry_client):
    app_id = "APEX-NARR03"
    company_id = "COMP-057"  # Technology, GROWTH, $1.1M requested
    
    # First session — crash after load_facts
    fraud_agent_1 = FraudDetectionAgent(event_store)
    fraud_agent_1._crash_after_node = "load_facts"  # Simulation hook
    
    state = {"application_id": app_id, "company_id": company_id, ...}
    
    try:
        await fraud_agent_1.process(state)
    except SimulatedCrashError:
        pass  # Expected
    
    # Check crashed session
    session_1_id = fraud_agent_1.session_id
    session_1_stream = await event_store.load_stream(f"agent-fraud-{session_1_id}")
    
    failed_event = next(e for e in session_1_stream if e.event_type == "AgentSessionFailed")
    assert failed_event.payload["recoverable"] is True
    assert failed_event.payload["last_successful_node"] == "load_facts"
    
    # Recovery session
    fraud_agent_2 = FraudDetectionAgent(event_store)
    fraud_agent_2._recover_from_session = session_1_id
    
    await fraud_agent_2.process(state)
    
    # Check recovery session
    session_2_id = fraud_agent_2.session_id
    session_2_stream = await event_store.load_stream(f"agent-fraud-{session_2_id}")
    
    started_event = next(e for e in session_2_stream if e.event_type == "AgentSessionStarted")
    assert started_event.payload["context_source"].startswith("prior_session_replay:")
    
    recovered_event = next(e for e in session_2_stream if e.event_type == "AgentSessionRecovered")
    assert recovered_event.payload["recovered_from_session_id"] == str(session_1_id)
    
    # Check no duplicate work
    all_node_events = [e for e in session_1_stream + session_2_stream 
                       if e.event_type == "AgentNodeExecuted"]
    load_facts_executions = [e for e in all_node_events if e.payload["node_name"] == "load_facts"]
    assert len(load_facts_executions) == 1, "load_facts executed exactly once across both sessions"
    
    # Check final output
    fraud_stream = await event_store.load_stream(f"fraud-{app_id}")
    assert any(e.event_type == "FraudScreeningCompleted" for e in fraud_stream)
```

### Implementation Requirements
- [`BaseApexAgent`](ledger/agents/base_agent.py) must support crash simulation hook
- [`AgentContextReconstructor.reconstruct_agent_context()`](ledger/integrity/gas_town.py) must identify last successful node
- Recovery session must skip completed nodes
- [`AgentSessionRecovered`](ledger/schema/events.py) event required in new session

## NARR-04: Compliance Hard Block (Montana)

### Scenario Description
[`ComplianceAgent`](ledger/agents/stub_agents.py) evaluates rules sequentially. REG-003 (Jurisdiction Lending Eligibility) fails because `company.jurisdiction = "MT"` (Montana excluded).

### Expected Behavior
1. REG-001 (Bank Secrecy Act) → [`ComplianceRulePassed`](ledger/schema/events.py)
2. REG-002 (OFAC Sanctions) → [`ComplianceRulePassed`](ledger/schema/events.py)
3. REG-003 (Jurisdiction) → [`ComplianceRuleFailed`](ledger/schema/events.py) with `is_hard_block = True`
4. **Evaluation stops** — REG-004, REG-005, REG-006 never evaluated
5. [`ComplianceCheckCompleted`](ledger/schema/events.py) with `overall_verdict = "BLOCKED"`
6. [`ApplicationDeclined`](ledger/schema/events.py) on loan stream with `adverse_action_notice_required = True`
7. **NO [`DecisionGenerated`](ledger/schema/events.py) event** ever appears

### Test Assertions
```python
# tests/test_narratives.py::test_narr04
async def test_narr04_montana_hard_block(event_store, registry_client):
    # Find the Montana company from seed data
    montana_company = await registry_client.get_company_by_jurisdiction("MT")
    app_id = "APEX-NARR04"
    company_id = montana_company["company_id"]
    
    # Run ComplianceAgent
    compliance_agent = ComplianceAgent(event_store)
    state = {"application_id": app_id, "company_id": company_id, ...}
    await compliance_agent.process(state)
    
    # Load compliance stream
    compliance_stream = await event_store.load_stream(f"compliance-{app_id}")
    
    # Count rule events
    rule_events = [e for e in compliance_stream if e.event_type in 
                   ["ComplianceRulePassed", "ComplianceRuleFailed", "ComplianceRuleNoted"]]
    
    assert len(rule_events) == 3, "Exactly 3 rules evaluated (REG-001, REG-002, REG-003)"
    
    # Check REG-003 failed with hard block
    reg003_event = next(e for e in rule_events if e.payload["rule_id"] == "REG-003")
    assert reg003_event.event_type == "ComplianceRuleFailed"
    assert reg003_event.payload["is_hard_block"] is True
    
    # Check overall verdict
    completed_event = next(e for e in compliance_stream if e.event_type == "ComplianceCheckCompleted")
    assert completed_event.payload["overall_verdict"] == "BLOCKED"
    
    # Check loan stream
    loan_stream = await event_store.load_stream(f"loan-{app_id}")
    
    # NO DecisionGenerated
    assert not any(e.event_type == "DecisionGenerated" for e in loan_stream)
    
    # ApplicationDeclined present
    declined_event = next(e for e in loan_stream if e.event_type == "ApplicationDeclined")
    assert "REG-003" in declined_event.payload["decline_reasons"]
    assert declined_event.payload["adverse_action_notice_required"] is True
```

### Implementation Requirements
- [`ComplianceAgent.build_graph()`](ledger/agents/stub_agents.py) must use conditional edges
- After each rule node, check `state.get("hard_block")`
- If `hard_block = True`, route to `write_output` (skip remaining rules)
- REG-003 implementation: `company.jurisdiction != "MT"`
- [`ComplianceAgent`](ledger/agents/stub_agents.py) is deterministic Python — **NO LLM in decision path**

## NARR-05: Human Override (The Loan Officer Approves Against the Agent)

### Scenario Description
Full pipeline runs on a DECLINING revenue company with high leverage. [`DecisionOrchestratorAgent`](ledger/agents/stub_agents.py) recommends DECLINE. A human loan officer overrides and approves.

### Expected Behavior
1. All agents run: Document → Credit → Fraud → Compliance → Decision
2. [`DecisionGenerated`](ledger/schema/events.py) with `recommendation = "DECLINE"` and `confidence = 0.82`
3. [`HumanReviewRequested`](ledger/schema/events.py) on loan stream
4. Human override via MCP tool: `record_human_review_completed`
5. [`HumanReviewCompleted`](ledger/schema/events.py) with:
   - `override = True`
   - `reviewer_id = "LO-Sarah-Chen"`
   - `final_decision = "APPROVE"`
   - `override_reason = "15-year customer, prior repayment history, collateral offered"`
6. [`ApplicationApproved`](ledger/schema/events.py) with:
   - `approved_amount_usd = 750000` (less than $950K requested)
   - `conditions = ["Monthly revenue reporting for 12 months", "Personal guarantee from CEO"]`

### Test Assertions
```python
# tests/test_narratives.py::test_narr05
async def test_narr05_human_override(event_store, registry_client, mcp_server):
    app_id = "APEX-NARR05"
    company_id = "COMP-068"  # Retail, 15-year customer, DECLINING -8% YoY
    
    # Run full pipeline
    # 1. DocumentProcessingAgent
    doc_agent = DocumentProcessingAgent(event_store)
    await doc_agent.process({"application_id": app_id, ...})
    
    # 2. CreditAnalysisAgent
    credit_agent = CreditAnalysisAgent(event_store, registry_client)
    await credit_agent.process({"application_id": app_id, "company_id": company_id, ...})
    
    # 3. FraudDetectionAgent
    fraud_agent = FraudDetectionAgent(event_store)
    await fraud_agent.process({"application_id": app_id, ...})
    
    # 4. ComplianceAgent
    compliance_agent = ComplianceAgent(event_store)
    await compliance_agent.process({"application_id": app_id, "company_id": company_id, ...})
    
    # 5. DecisionOrchestratorAgent
    orchestrator = DecisionOrchestratorAgent(event_store)
    await orchestrator.process({"application_id": app_id, ...})
    
    # Check DecisionGenerated
    loan_stream = await event_store.load_stream(f"loan-{app_id}")
    decision_event = next(e for e in loan_stream if e.event_type == "DecisionGenerated")
    assert decision_event.payload["recommendation"] == "DECLINE"
    
    # Check HumanReviewRequested
    assert any(e.event_type == "HumanReviewRequested" for e in loan_stream)
    
    # Human override via MCP
    result = await mcp_server.call_tool("record_human_review_completed", {
        "application_id": app_id,
        "reviewer_id": "LO-Sarah-Chen",
        "final_decision": "APPROVE",
        "override": True,
        "override_reason": "15-year customer, prior repayment history, collateral offered",
        "approved_amount_usd": 750000,
        "conditions": [
            "Monthly revenue reporting for 12 months",
            "Personal guarantee from CEO"
        ]
    })
    
    # Reload loan stream
    loan_stream = await event_store.load_stream(f"loan-{app_id}")
    
    # Check HumanReviewCompleted
    review_event = next(e for e in loan_stream if e.event_type == "HumanReviewCompleted")
    assert review_event.payload["override"] is True
    assert review_event.payload["reviewer_id"] == "LO-Sarah-Chen"
    assert review_event.payload["final_decision"] == "APPROVE"
    
    # Check ApplicationApproved
    approved_event = next(e for e in loan_stream if e.event_type == "ApplicationApproved")
    assert approved_event.payload["approved_amount_usd"] == 750000
    assert len(approved_event.payload["conditions"]) == 2
```

### Implementation Requirements
- All 5 agents must be fully implemented
- MCP tool `record_human_review_completed` must exist
- Command handler for [`HumanReviewCompleted`](ledger/schema/events.py) must append [`ApplicationApproved`](ledger/schema/events.py)
- This is the application used for `scripts/demo_narr05.py` (must run < 90 seconds)
